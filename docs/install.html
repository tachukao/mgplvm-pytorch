InstallÔÉÅ

The latest version of this library is in the develop branch of the mGPLVM repository. You can install the latest version by running the following:

git clone https://github.com/tachukao/mgplvm-pytorch
cd mgplvm-pytorch
git checkout develop
pip install -e .
